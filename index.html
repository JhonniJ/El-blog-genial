Git master, la guia que necesitabas para dominar el mundo remoto

Seguro te preguntas ¿qué es git? y ¿porqué deberia aprenderlo?, bueno para ser simples git es un sistema que te permite almacenar los cambios del proyecto con detalles y la posibilidad de volver a versiones anteriores, esto es lo que se conoce como sistema de control de versiones. Para comenzar en el mundo de git hay 2 formas de trabajar con el, la forma local y la forma remota, git es una tecnologia que trabaja de forma nativa pero también hay servicios como github o gitlab que usando servidores en la nube permiten trabajar de forma remota.

En esta ocación vamos a trabajar de forma local

Preparación

Antes de trabajar con git hay que instalarlo, para hacerlo nos dirijimos a la página oficial https://git-scm.com/ en ella encontraremos la opción descargar que por suerte la página entendera si somos de windows o mac, una vez hecho esto solo lo instalamos pero tenemos que estar pendiente de activar 2 cosas fundamentales, la primera, la interfaz de gitbash, es una consola especial y la segunda que esta vinculada con esta es el guardado avanzado, esto lo que permite es usar los comandos tanto de windows como mac sin inconvenientes

Dominando la consola

Estamos casi listos para comenzar, puede parecer mucho o muy complicado pero tranquilo no es tan dificil y nada que valga la pena es fácil, en esta parte vamos a ver comandos para navegar en la linea de comandos de git, el tan afamado y nuestro nueva compañero, Gitbash < b> Comandos para moverse
- pwd - > muestra en donde Estamos
- cd - > navegar entre carpetas
- ls - > muestra los archivos
